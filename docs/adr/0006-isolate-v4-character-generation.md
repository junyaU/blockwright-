# v4キャラ生成の非決定論を1段に隔離し、ローカル個人利用限定とする

v4（喋るだけでキャラ生成）は本プロジェクト唯一の「AIが形に触れる」例外であり、その非決定論を**stage3（image→3D生成）の1段だけに封じ込める**。前段は言語・画像取得（SerpAPI）、stage4（cleanup）で荒さを吸収、stage5以降は決定論、破綻時は v3 平面建築にフォールバックする。荒さを放置すると下流すべてが汚染されるため、不確実性は避けるのではなく隔離して飼い慣らす。

## 制約（コードに見えない前提）

- **ローカル個人利用限定。** SerpAPI による画像取得・第三者キャラクターIPの扱いの都合上、共有・配布しない。`SERPAPI_API_KEY` 未設定なら「作って」はパラメトリック経路のみに退避する。
- これは [0005](0005-parametric-first-generation-last-resort.md) の「生成は逃げ道」を、原則を破ってでも成立させるための封じ込め設計である。
