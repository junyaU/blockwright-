# AIは設計、コードは施工

AIが出してよいのは**意図・パラメータ・構造化された中間表現（IR）と分類**だけとし、**座標・幾何・voxel占有は決定論コード**が生成する。AIに生 `setblock`／`fill`／絶対座標／voxel占有を出させない。LLMは精密な3D座標計算と逐次的な状態追跡が苦手で、ここを任せると壁ズレ・宙ぶらりん・左右非対称・回路の沈黙が出るため。

## 検討した選択肢
